## Clone this repository
```bash
git clone https://github.com/j-schneble/jsdev-personal.git
```
## Install dependencies
cd client && server
```bash
npm install
```
## CLIENT on [Localhost:3000] SERVER on [Localhost:8000]
```bash
npm run dev
```
